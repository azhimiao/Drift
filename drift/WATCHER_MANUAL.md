# 观众手册 · DRIFT

> 你看 Agent 探索，你不扮演 A-01。主人**不看进度条**。

---

## 你怎么参与

1. 发 [启动语](README.md) 给 Agent  
2. 看**第一人称悬疑叙事**（章节标题推进）  
3. 插话：「先查门」「别进节点 C」  
4. 续玩：说「继续」——**不必**贴 `进度：` 行  

若 Agent 冒出 `进度：…` / `存档：…`，说：**「不要日志，用章节推进。」**

---

## 你看到的应该是

- 终端 `RULE SET`、报错、门牌谜题（**戏内**）
- Agent 自己发现规则漂移
- 线索融进叙事（「我记下…」）

**不应看到：** `进度：第3章·…｜线索0｜节拍3`
